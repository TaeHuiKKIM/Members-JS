<h1>HTML / CSS / JS로 개발하여 운영중인 반응형 디자인의 병원 웹사이트</h1>

멤버스 성형외과
https://membersclinic.com

서버나 큰 기능 없이 병원을 소개하며 웹사이트에 방문한 고객이 카카오 채널 문의로 넘어가도록 유도하였습니다.

모바일 버전에서 호환되도록 반응형으로 디자인하고 웹에 쉽게 노출될 수 있게 이미지 태그에 하나하나 설명을 넣거나 이미지 용량을 줄이는 등 SEO를 입문하며 개발하였습니다.

병원에서는 별다른 이미지 제공이 없어 20년전에 사용됐던 저화질의 사진들을 업스케일링하여 재활용하였고, 이미지 일부를 텍스트화 시켰습니다.

첫 프로젝트이고 기획, 디자인, 개발, 배포까지 혼자 힘으로 해냈다는 점에서 의의가 있는 프로젝트이고 추후 Spring 기반으로 고도화 예정입니다.
